костюм "Горка - Tiger"

Комплектация

Костюм состоит из куртки и брюк.

  • КУРТКА (Горка - Tiger)
  • - Основной материал — палаточная ткань высокой прочности.
  • - Усиленные накладки из ткани "oxford rip-stop" в плечевой области.
  • - Дополнительные накладки на рукавах в области локтей.
  • - Манжеты с встроенными резинками выполнены из ткани oxford rip-stop.
  • - В области талии по бокам вшиты резинки.
  • - Внизу куртки проходит утяжка с фиксаторами.
  • - В предплечье вшиты резинки для утяжки.
  • - Капюшон с регулировкой в двух измерениях: по овалу лица и регулировкой бокового зрения.
  • - Внизу куртки два кармана с клапанами на пуговицах.
  • - По одному карману на липучке на каждом рукове куртки.
  • БРЮКИ (Горка - Tiger)
  • - Основной материал — палаточная ткань высокой прочности.
  • - Накладки из ткани "oxford rip-stop" (колени, седалище и голени)
  • - Ширинка на молнии.
  • - В комплекте специальные подтяжки, крепящиеся по бокам на защелки-фастекс.
  • - Широкие шлевки для ремня.
  • - Карманы: 2 прорезных кармана по бокам, 2 набедренных грузовых кармана с клапанами, 2 задних кармана на пуговицах.
  • - В области талии и голени вшиты утягивающие резинки.
  • - Пыльники для защиты от насекомых по низу штанин.
